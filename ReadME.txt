*Os três pilares do bom desenvolvedor
**Foco
	-> Escolher uma Stack e seguir com ela até o fim para se tornar melhor com ela, assim, evitando ficar migrando em varias linguagens, após domina-la sinta-se livre para migrar, com isso evita ficar desperso tentando aprender tudo ao mesmo tempo, até porque nenhum Dev sabe tudo...

**Prática
	-> Praticar sempre, estudar sempre, só assim para conseguir dominar o tão almejado sonho de se tornar um bom desenvolvedor...

**Grupo
	-> Estar em grupos de desenvolvedores, estar proximo de pessoas que vivem no mesmo ambiente, com isso as pessoas se ajudam, tiram duvidas, aprendem juntos, evitar ter medo de perguntar. Networking é fundamental... SEJA CRIATIVO
